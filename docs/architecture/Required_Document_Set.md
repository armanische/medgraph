# Required Document Set

**Status:** MVP-019 architecture baseline  
**Scope:** minimum document expectations for Golden Dataset v1

## Purpose

The Required Document Set defines the evidence package needed before a product
can move from discovery into serious review. The package differs by category.

Documents are required for review readiness, not automatic publication.

## Document Tiers

- Required: must exist before the product can be considered review-ready.
- Preferred: strongly recommended for complete review.
- Additional: useful context, never a replacement for required documents.

## Аппарат ИВЛ

Required:

- РУ / regulator record;
- IFU or user manual;
- datasheet or technical specification.

Preferred:

- service manual;
- alarms and modes documentation;
- accessories and consumables list.

Additional:

- brochure;
- training guide.

## Монитор пациента

Required:

- РУ / regulator record;
- IFU or user manual;
- datasheet with measured parameters.

Preferred:

- module compatibility guide;
- service manual;
- alarm documentation.

Additional:

- brochure;
- connectivity guide.

## Наркозная станция

Required:

- РУ / regulator record;
- IFU or user manual;
- datasheet;
- ventilation and vaporizer compatibility documentation.

Preferred:

- service manual;
- accessories and breathing circuit compatibility list.

Additional:

- brochure;
- training guide.

## УЗИ

Required:

- РУ / regulator record;
- user manual or IFU;
- datasheet;
- probe list or transducer compatibility document.

Preferred:

- service manual;
- software/options list.

Additional:

- brochure;
- clinical application guide.

## Эндоскопия

Required:

- РУ / regulator record;
- IFU;
- datasheet;
- compatibility document for processors, monitors or accessories.

Preferred:

- reprocessing/sterilization instructions when reusable;
- service manual when applicable.

Additional:

- brochure;
- quick reference guide.

## Операционное освещение

Required:

- РУ / regulator record;
- user manual;
- datasheet.

Preferred:

- installation manual;
- service manual.

Additional:

- brochure;
- room planning guide.

## Инкубаторы и неонатальное оборудование

Required:

- РУ / regulator record;
- IFU or user manual;
- datasheet with environmental and alarm characteristics.

Preferred:

- service manual;
- accessories list;
- cleaning and maintenance instructions.

Additional:

- brochure;
- training guide.

## Инфузионные и шприцевые насосы

Required:

- РУ / regulator record;
- IFU or user manual;
- datasheet.

Preferred:

- service manual;
- compatible syringes or infusion sets list;
- drug library/software documentation when applicable.

Additional:

- brochure;
- quick start guide.

## Аспираторы

Required:

- РУ / regulator record;
- user manual;
- datasheet.

Preferred:

- service manual;
- accessory and consumable list.

Additional:

- brochure.

## Safety Rules

Never:

- replace IFU with brochure for critical characteristics;
- use dealer catalog as the main document package;
- treat file name as document type without review;
- publish compatibility without compatibility evidence;
- mark product complete when required documents are missing.
